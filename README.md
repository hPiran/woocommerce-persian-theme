# 🛒 WooPersian Store — قالب ووکامرس فارسی

قالب حرفه‌ای فروشگاهی ووکامرس با پشتیبانی کامل **RTL** و زبان **فارسی**.

## ✨ ویژگی‌ها

- 🇮🇷 **فارسی کامل** — تمام متون رابط کاربری به فارسی
- 🔄 **RTL حرفه‌ای** — `rtl.css` اختصاصی + CSS logical properties
- 🔤 **فونت Vazirmatn** — از Google Fonts
- 🔢 **اعداد فارسی** — نمایش قیمت‌ها با اعداد فارسی (۱, ۲, ۳)
- 📱 **ریسپانسیو** — موبایل، تبلت، دسکتاپ
- 🛒 **مینی‌کارت AJAX** — بروزرسانی زنده بدون رفرش
- 🎨 **کاستومایزر** — تنظیم رنگ‌ها، سربرگ، فوتر از پیشخوان
- ⚡ **بهینه** — بدون jQuery (JS خالص Vanilla)
- 🔒 **امن** — esc_html/esc_attr در تمام خروجی‌ها

## 📦 نصب

1. فایل ZIP رو از [Releases](../../releases) دانلود کنید
2. در پیشخوان وردپرس → **نمایش** → **پوسته‌ها** → **افزودن** → **بارگذاری پوسته**
3. فایل ZIP رو آپلود و نصب کنید
4. پوسته رو فعال کنید

یا از طریق Git:
```bash
cd wp-content/themes/
git clone https://github.com/hPiran/woocommerce-persian-theme.git
```

## 🎨 تنظیمات

بعد از فعال‌سازی:

1. **منوها**: پیشخوان → نمایش → منوها → منوی اصلی رو بسازید
2. **ابزارک‌ها**: پیشخوان → نمایش → ابزارک‌ها → سایدبار و فوتر رو تنظیم کنید
3. **کاستومایزر**: پیشخوان → نمایش → سفارشی‌سازی → رنگ‌ها و تنظیمات سربرگ
4. **ووکامرس**: صفحات فروشگاه، سبد خرید و تکمیل خرید رو از تنظیمات ووکامرس بسازید
